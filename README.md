A beautiful, interactive Valentine's Day webpage with background music, animated elements, and playful interactions.

## Features

- **Interactive Question**: "Will you be my Valentine?" with Yes/No buttons
- **Playful "No" Button**: The "No" button moves away when you try to click it
- **Image Transitions**: Images change based on user interaction
- **Background Music**: Taylor Swift's "Timeless" plays in the background
- **Music Controls**: 
  - Play/Pause button with visual feedback
  - Progress slider to seek through the song
  - Real-time time display (current time / total duration)
- **Animated Background**: Floating hearts animation
- **Glassmorphism Design**: Modern UI with purple/pink gradients
- **Responsive Design**: Works beautifully on desktop and mobile devices
- **Smooth Animations**: Pulse effects, hover states, and transitions throughout

## Files

- `index.html` - Main webpage with all functionality
- `taylor_swift_timeless.mp3` - Background music
- `nnns_looking_away.jpg` - Initial image
- `nnns_looking_at_each_others.jpg` - Success state image

## Usage

Simply open `index.html` in a web browser. The music will attempt to auto-play (browser dependent) or will start on first user interaction.

## Browser Compatibility

Works best in modern browsers (Chrome, Firefox, Safari, Edge). Note that some browsers may block autoplay - just click anywhere on the page or the play button to start the music.
